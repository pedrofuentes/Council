Join the community and help make Council better.

We welcome contributions of all kinds: bug reports, feature requests, code
improvements, documentation edits, example panels, and more. Council is
open-source and thrives on community participation.

**Getting started:**
- Report bugs or suggest features via [GitHub Issues](https://github.com/pedrofuentes/Council/issues)
- Improve documentation (see `CONTRIBUTING.md` for workflow)
- Submit example panels that showcase new use cases
- Contribute code (see development workflow and testing guidelines)

This project follows test-driven development and requires Sentinel review for
all changes. See `AGENTS.md` and `docs/SENTINEL.md` for the complete workflow.

_Comprehensive contributing guides will be added in a future update._
