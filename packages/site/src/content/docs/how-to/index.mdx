Solve specific problems with practical, task-focused guides.

**How-to guides** are problem-oriented — they assume you understand the basics
and want to accomplish a particular goal efficiently. Each guide focuses on one
concrete task with minimal explanation.

Use these when you know *what* you want to do and need the fastest path to get
it done — whether that's configuring tool permissions, customizing expert
personas, exporting panel transcripts, or integrating with your CI pipeline.

_Comprehensive how-to guides will be added in a future update._
