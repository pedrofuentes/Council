Council is a CLI tool that assembles persistent AI expert panels for
deliberation and decision-making. Experts collaborate, challenge each other's
reasoning, and build on shared context across sessions.

This documentation is organized using the [Diátaxis](https://diataxis.fr/)
framework to help you find what you need efficiently:

- **[Tutorials](/Council/docs/tutorials/)** — Learn by building complete examples from first principles
- **[How-To Guides](/Council/docs/how-to/)** — Solve specific tasks with practical recipes
- **[Reference](/Council/docs/reference/)** — Look up technical details, commands, and schemas
- **[Explanation](/Council/docs/explanation/)** — Understand concepts, architecture, and design decisions

## Getting Started

New to Council? Start with the **Tutorials** section to learn core concepts
through hands-on examples. Already familiar with the basics? Jump to **How-To
Guides** for task-focused solutions.

Council works with any LLM provider that supports structured outputs — Copilot,
OpenAI, Anthropic, and more. Documentation uses provider-agnostic patterns
wherever possible.
